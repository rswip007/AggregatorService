# Aggregator Service

Aggregator Service is a Kotlin application that provides various statistics and insights from football match data.  

The application expects a csv file stored in the resources folder in the csv format. However the path can be changed in the main class.

## Features

- Generate a league table with team statistics.
- Get detailed information about a specific match.
- Get game scores for a specific match.
- Get player statistics.
- Get top 'n' scorers.
- Get 'n' players with the most assists.

## Prerequisites

- Java 8 or higher
- Gradle

## Installation

Clone the repository:

```bash
git clone
```

Build the project:

```bash
./gradlew build
```

## Usage

Run the application:

```bash
./gradlew run
```
